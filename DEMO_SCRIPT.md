## 功能演示提纲（按时间顺序追加）

### 1. AI 多轮对话
- **一句话描述**：基于 Kimi K2.6 的流式对话，支持多轮上下文、图片理解、联网搜索。
- **演示要点**：发送消息 → 展示流式输出 → 展示历史上下文连续性。
- **预估耗时**：20 秒

### 2. 结构化数据输出（Generative UI）
- **一句话描述**：通过 JSON Mode 强制 AI 输出结构化数据，前端自动渲染为交互式卡片、雷达图和时间轴。
- **演示要点**：
  1. 切换角色为「简历分析」，输入一段简历文本 → AI 返回 JSON，前端渲染为「简历卡片」（含头像、技能标签、薪资）。
  2. 切换角色为「心理测试」，输入心理状态描述 → AI 返回 JSON，前端渲染为「评分仪表盘 + SVG 雷达图 + 维度进度条」。
  3. 切换角色为「旅游规划」，输入目的地和天数 → AI 返回 JSON，前端渲染为「时间轴行程单」（含时间、景点、预算、贴士）。
- **预估耗时**：60 秒

### 3. Artifacts 独立工作区（Side-by-side Canvas）
- **一句话描述**：当 AI 生成长代码或长文章时，右侧自动滑出独立 Canvas 面板，支持语法高亮、复制、展开/收缩。
- **演示要点**：
  1. 让 AI 写一段超过 20 行的代码（如快速排序）→ 消息hover时出现「Canvas」按钮 → 点击后右侧滑出代码面板。
  2. 展示面板内的语法高亮、一键复制、展开/收缩功能。
  3. 关闭面板后回到正常会话列表，再次打开其他长内容可切换。
- **预估耗时**：30 秒

### 4. 全局个性化记忆（Custom Instructions）
- **一句话描述**：用户可设置「关于我」和「回复偏好」，系统自动拼接到每条请求的 system prompt 中，实现千人千面。
- **演示要点**：
  1. 点击顶部齿轮图标 → 填写「关于我：我是一个 5 岁的小孩」「回复偏好：请用童话故事解释」→ 保存。
  2. 问一个复杂的物理问题（如「什么是黑洞」）→ AI 立刻用童话口吻回答。
  3. 新建会话，再次提问 → 偏好仍然生效，证明是全局记忆而非会话级。
- **预估耗时**：30 秒
