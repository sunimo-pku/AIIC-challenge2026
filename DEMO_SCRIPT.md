## 功能演示提纲（按时间顺序追加）

### 1. AI 多轮对话
- **一句话描述**：基于 Kimi K2.6 的流式对话，支持多轮上下文、图片理解、联网搜索。
- **演示要点**：发送消息 → 展示流式输出 → 展示历史上下文连续性。
- **预估耗时**：20 秒

### 2. 结构化数据输出（Generative UI）
- **一句话描述**：通过 JSON Mode 强制 AI 输出结构化数据，前端自动渲染为交互式卡片、雷达图和时间轴。
- **演示要点**：
  1. 切换角色为「简历分析」，输入一段简历文本 → AI 返回 JSON，前端渲染为「简历卡片」（含头像、技能标签、薪资）。
  2. 切换角色为「心理测试」，输入心理状态描述 → AI 返回 JSON，前端渲染为「评分仪表盘 + SVG 雷达图 + 维度进度条」。
  3. 切换角色为「旅游规划」，输入目的地和天数 → AI 返回 JSON，前端渲染为「时间轴行程单」（含时间、景点、预算、贴士）。
- **预估耗时**：60 秒

### 3. Artifacts 独立工作区（Side-by-side Canvas）
- **一句话描述**：当 AI 生成长代码或长文章时，右侧自动滑出独立 Canvas 面板，支持语法高亮、复制、展开/收缩。
- **演示要点**：
  1. 让 AI 写一段超过 20 行的代码（如快速排序）→ 消息hover时出现「Canvas」按钮 → 点击后右侧滑出代码面板。
  2. 展示面板内的语法高亮、一键复制、展开/收缩功能。
  3. 关闭面板后回到正常会话列表，再次打开其他长内容可切换。
- **预估耗时**：30 秒

### 4. 全局个性化记忆（Custom Instructions）
- **一句话描述**：用户可设置「关于我」和「回复偏好」，系统自动拼接到每条请求的 system prompt 中，实现千人千面。
- **演示要点**：
  1. 点击顶部齿轮图标 → 填写「关于我：我是一个 5 岁的小孩」「回复偏好：请用童话故事解释」→ 保存。
  2. 问一个复杂的物理问题（如「什么是黑洞」）→ AI 立刻用童话口吻回答。
  3. 新建会话，再次提问 → 偏好仍然生效，证明是全局记忆而非会话级。
- **预估耗时**：30 秒

### 5. 局部重写 / 文本润色 (Inline Edit)
- **一句话描述**：用户在 AI 回复上划选任意段落，弹出悬浮菜单进行局部修改，模型只重写选中的部分并原地替换。
- **演示要点**：
  1. 让 AI 生成一段文字（如一封邮件草稿）。
  2. 选中其中一段文字 → 弹出「更正式 / 更简短 / 扩写 / 自定义」菜单。
  3. 点击「更正式」→ 等待 1-2 秒 → 选中的文本被替换为更正式的表达，其余内容不变。
  4. 再选中另一段 → 点击「自定义」→ 输入"加入数据支撑" → 文本被替换。
- **预估耗时**：30 秒

### 6. 泛用函数调用框架 (Function Calling)
- **一句话描述**：AI 不再只是"说话的嘴"，而是能主动调用本地 Python 工具完成实际任务（查天气、查库存等）。
- **演示要点**：
  1. 打开左侧「工具 ON」开关。
  2. 输入"北京今天天气怎么样？" → AI 自动调用 `get_weather` 工具 → 返回"晴，25度" → AI 组织成自然语言回复。
  3. 输入"苹果手机还有货吗？多少钱？" → AI 自动调用 `query_inventory` 工具 → 返回库存和价格 → AI 组织成结构化回复。
  4. （可选）强调：比赛日只需在 `agent_tools.py` 里加 Python 函数，前端零改动。
- **预估耗时**：30 秒

### 7. MockMate AI 模拟面试官 — 面试设置与面试攻略
- **一句话描述**：用户输入目标公司和岗位，上传 PDF 简历，AI 自动生成定制化面经报告。
- **演示要点**：
  1. 左侧栏选择「字节跳动」+「后端开发」→ 上传简历 PDF → 保存。
  2. 进入「面试攻略」→ AI 基于联网搜索生成结构化面经（公司背景、岗位 JD、面试流程、高频题）。
  3. 展示右侧动态面板中的面经报告排版。
- **预估耗时**：30 秒

### 8. 五关面试流程（简历评估 → 总结）
- **一句话描述**：模拟真实大厂面试流程，每关有不同面试官角色和考察重点。
- **演示要点**：
  1. 点击顶部导航栏自由跳转到「简历评估」→ AI 提取技术标签、风险点、深挖项目。
  2. 跳转到「技术面」→ AI 交替进行八股问答和项目深挖 → 右侧雷达图实时评分。
  3. 跳转到「情景面」→ AI 现场生成冲突场景 + STAR 行为面试。
  4. （本地 localhost）展示语音输入功能。
  5. 强调：支持非线性闯关，随时可回退复习。
- **预估耗时**：45 秒
